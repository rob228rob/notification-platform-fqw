# Платформа уведомлений

Рабочее пространство описывает контейнерную архитектуру платформы уведомлений. Основной поток проходит через командный фасад, Kafka, delivery-dispatcher, сервисы канальной отправки и history-writer.

Диаграммы разделены по назначению: общий состав сервисов, входной поток, Kafka-поток, обработка email-уведомления, маршрутизация доставки и наблюдаемость.
