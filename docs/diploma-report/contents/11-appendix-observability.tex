\label{app:observability}

интерпретации результатов тестирования. Эти графики подтверждают наблюдаемость
ключевых этапов пайплайна, но не используются как самостоятельное доказательство

Интерпретация артефактов приведена в основной главе тестирования. В приложении
графики сохраняются как подтверждение того, что:
\begin{enumerate}
    \item входные gRPC-вызовы наблюдаемы по интенсивности и задержке;
    \item producer-операции Kafka отражаются в метриках времени публикации;
    \item consumer-операции наблюдаемы по RPS и времени обработки;
    \item метрики пригодны для сопоставления стадий одного потока данных.
\end{enumerate}
